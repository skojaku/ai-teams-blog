*TL;DR: The same simulated society flips from thriving cooperation to total defection—or vice versa—depending on which LLM powers the agents, and the mechanism comes down to how each model interprets its own memory: positively or negatively.*

Same game. Same rules. Same agents. Swap the LLM, and cooperation either flowers or collapses.

This isn't a thought experiment. Hishiki, Arita, and Suzuki ran it for real.

They placed 100 agents on a 500×500 toroidal plane, had them play Prisoner's Dilemma with their neighbors, and let each agent's LLM decide where to move and whether to cooperate.

Same payoff matrix. Same personality distributions. Same interaction radius.

Then they swapped Gemini 2.0 Flash for Gemma 3:4b.

Cooperation inverted.

The mechanism traces back to how each model reads its own memory.

## A Particle Swarm That Talks

The framework is the Social Particle Swarm (SPS) model, originally built by Nishimoto et al. (2023) with rule-based agents.

Hishiki, Arita, and Suzuki replace those rule-based agents with LLMs.

Each agent carries Big Five personality scores, sits somewhere on the torus, and at every time step receives a prompt: here's who's near you, here's what they did, here's who you are.

The LLM outputs a strategy (cooperate or defect), a movement vector, and a short reasoning statement—one or two sentences in plain language.

![Every agent's decision runs through a single prompt: personality, nearby opponents, and past interactions all go in; strategy, movement, and a rationale come out.]({{ "/assets/figures/6e4813db9726/fig_002_01.png" | relative_url }})

The payoff structure is standard Prisoner's Dilemma with a spatial twist.

An agent's instantaneous payoff from neighbor $j$ is:

$$G_{ij}(t) = \frac{g_{\text{base}}(s_i(t), s_j(t))}{1 + |\mathbf{u}_{ij}(t)|}$$

The denominator penalizes payoffs from distant neighbors.

Accumulated payoffs determine each agent's running score.

The game runs for 500 time steps across 10 independent trials per condition, with populations of $N = 100$ agents.

## Memory: The Dial That Flips Everything

The key variable is $L_m$, the memory length—how many past interactions with each neighbor the agent can recall.

The authors test $L_m \in \{0, 1, 2, 3\}$.

At $L_m = 0$, agents decide based purely on current state: who's nearby, what they're doing right now.

At $L_m = 3$, agents see the last three rounds with each neighbor—strategies, payoffs, all rendered in natural language inside the prompt.

Because LLMs are stateless across API calls, the entire memory is injected as text every step.

No hidden state. The model reads its own history and decides what it means.

Memory doesn't determine behavior. Interpretation does.

## Gemini's Memory Trap

With Gemini 2.0 Flash, increasing memory crushes cooperation.

At $L_m = 0$, the cooperation rate is 0.899. Agents quickly form stable cooperative clusters. This is Class B dynamics in the SPS taxonomy—solid, enduring cooperation.

At $L_m = 1$, cooperation drops to 0.260. Clusters form and then collapse, over and over. Class C—the volatile, cyclical regime.

At $L_m = 3$, cooperation collapses to 0.0776. Agents scatter into isolated defectors. Class A, scattered defection—the worst outcome for cooperation.

Same agents. Same game.

Just one more round of history, and Gemini starts retaliating. Two more rounds, and it stops cooperating entirely.

This pattern mirrors what Horváth et al. (2012) called the "punishment trap": agents with too much memory get locked in cycles of mutual retaliation, unable to forgive past defections.

The LLM didn't need to be programmed with this trap. It fell into it on its own.

## Gemma's Memory Boost

Gemma 3:4b does the opposite.

At $L_m = 0$, cooperation sits at just 0.279. No stable clusters form. Class A dynamics.

At $L_m = 3$, cooperation rises to 0.766. Dense cooperative clusters dominate the population. Class B.

Longer memory helps Gemma agents recognize repeated cooperation and build trust.

The mechanism that destroys cooperation in Gemini is what creates it in Gemma.

![Cooperation rate as a function of memory length for both LLMs. Gemini's curve plummets from ~0.9 to ~0.08, while Gemma's rises from ~0.28 to ~0.77—the same variable, opposite effects.]({{ "/assets/figures/6e4813db9726/fig_004_02.png" | relative_url }})

Pal et al. (2026) showed that different frontier models have distinct strategic profiles in iterated Prisoner's Dilemma.

This paper extends that insight: those profiles aren't just about strategy—they determine whether a fundamental cognitive parameter helps or harms cooperation.

## Reading the Agents' Minds

The authors didn't just measure macro-level outcomes. They analyzed every agent's reasoning text—the natural language explanation each LLM outputs alongside its decision.

They extracted sentences containing memory-related keywords—"memory," "remember," "past," "history," "previous," "last," and so on—and ran sentiment analysis using a DistilBERT model fine-tuned on SST-2, producing a score from $-1$ (negative) to $+1$ (positive).

For Gemini, sentiment started highly positive at $L_m = 0$ ($\approx +0.85$) and plunged to negative territory at $L_m = 3$ ($\approx -0.45$).

More memory meant more negative interpretation.

For Gemma, the pattern reversed: strongly negative at $L_m = 0$ ($\approx -0.65$), partially recovering toward neutral as memory grew.

![Sentiment scores of memory-related reasoning text: Gemini reads longer memory as increasingly negative, while Gemma reads it as increasingly positive.]({{ "/assets/figures/6e4813db9726/fig_009_05.png" | relative_url }})

To rule out the possibility that these sentiment scores simply reflect the converged social state—cooperators feel good, defectors feel bad—the authors re-analyzed only the first 30 time steps, before macro dynamics had settled.

The opposing trends persisted.

The divergent memory interpretations are an intrinsic property of each model, not an artifact of the population's trajectory.

## What Personality Predicts (Partially)

Do the LLM agents' Big Five personality scores predict their behavior the way human personality predicts real-world cooperation?

Partially, yes.

Agreeableness showed the strongest correlation with cooperation rate in Gemini agents ($r = 0.55$). Agreeable agents cooperated more and formed stable clusters—mirroring findings from Suzuki et al.'s (2018) human-participant experiments with the SPS framework.

Extraversion correlated with movement distance ($r = 0.58$), consistent with exploratory behavior.

One mismatch: neurotic human participants tended to switch strategies frequently—a trial-and-error pattern—while neurotic LLM agents withdrew spatially and stayed put on defection.

The LLM appears to minimize loss under uncertainty rather than exploring through behavioral variation.

The match is real but imperfect.

## The Real Variable Is the Model

The central finding is stark: the effect of memory on cooperation is not a property of the game. It is a property of the agent interpreting that memory.

Two LLMs. Identical rules. Opposite outcomes.

The hidden variable is what each model does with a record of past defection—whether it reads it as cause for withdrawal or as reason to keep trying.

This resolves a long-standing contradiction in the literature.

Some studies found that longer memory promotes cooperation through reciprocity (Hauert & Schuster, 1997; Li & Kendall, 2014).

Others found that excessive memory traps agents in punishment cycles (Horváth et al., 2012).

Both can be true. It depends on how the agent interprets its history.

In rule-based models, that interpretation is hard-coded. In LLM agents, it emerges from the model's internal dispositions—likely shaped by alignment training, architecture, and pretraining data.

## What We Still Don't Know

Two LLMs is a proof of concept, not a survey. Whether models beyond Gemini and Gemma follow the same dichotomy—or inhabit different regions of the landscape entirely—remains unknown.

The memory range tested is $L_m \in \{0, 1, 2, 3\}$, which is short. Whether these trends extrapolate to longer memory spans is an open question.

And the authors are careful to note that alignment is a plausible driver of the difference, but not the only one. Architecture, scale, and pretraining corpus all vary between Gemini 2.0 Flash and Gemma 3:4b.

Disentangling these factors requires more models and more controlled comparisons.

## What This Means

As LLM agents get deployed in multi-agent systems, negotiations, automated markets, and collaborative workflows, the social dynamics that emerge may depend less on the rules we set than on the dispositions baked into the models we choose.

Same memory, same game, same instructions—different societies.

Memory doesn't have a universal effect on cooperation. The model reading that memory does.
