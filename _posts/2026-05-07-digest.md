## Themes

**Shared state needs selection, not just storage.** Every system today accumulates — chat logs, memory files, reward signals — and treats accumulation as coordination. Three of today's papers explicitly reject that assumption. *Governed Collaborative Memory as Artificial Selection in LLM-Based Multi-Agent Systems* argues that deciding which memories become durable shared state is a selection regime, not a storage problem. *Evolving Idea Graphs with Learnable Edits-and-Commits for Multi-Agent Scientific Ideation* replaces the chat log with an editable graph where claims are visible, flaggable, and repairable. *Tree-based Credit Assignment for Multi-Agent Memory System* branches the reward pipeline so each agent receives its own credit signal instead of sharing the final score. The shared substrate is where the hardest design decision lives, and none of these papers leave that decision on autopilot.

**Knowing when not to act is the competitive advantage.** *Uno-Orchestra: Parsimonious Agent Routing via Selective Delegation* trains a 7B router that learns both whether to decompose a task and where to route it — and the cost savings come almost entirely from questions it skips. *Evolving Idea Graphs* gives its controller a commit head that learns to stop refining when the graph is ready, not when a fixed round count expires. *Governed Collaborative Memory* names four selection regimes for shared memory, each making a different bet on which traits can be safely automated and which need explicit judgment. Parsimony is not a constraint. It is a learned skill.

**Failure that persists is either poison or infrastructure — depending on who can see it.** *When Stress Becomes Signal* shows that every multi-agent architecture loses roughly a third of its quality under semantic stress, but the stress also produces structured, convex-expansive variation. *Governed Collaborative Memory* documents a case where a fabricated analysis entered shared memory and was later converted into a ratified principle — governance did not prevent the error, it turned the error into future selection criteria. *Tree-based Credit Assignment* isolates each agent's contribution by branching the pipeline, making failure traceable to a role rather than smeared across the team. The difference between a bug and a design lesson is whether the system can observe, attribute, and learn from what went wrong.

## Editorial

All five papers land on the same problem from different angles: multi-agent LLM systems are coordinated by substrates nobody is minding. The chat log absorbs contradictions. The shared reward smears credit. The memory file persists whatever an agent writes. The stress measurement stops at quality drop. Every system treats its shared medium as a passive pipe, not a design decision.

What distinguishes this batch is that none of the fixes are bigger models. They are better boundaries. TreeMem branches the pipeline to isolate who did what. Governed Memory adds a ratification checkpoint before anything becomes durable. Uno-Orchestra hides worker identities so the router can't take shortcuts. Evolving Idea Graphs makes the idea itself the shared object, not the transcript about the idea. CAFE measures the geometry of stress so future systems can route the right stress to the right architecture.

The incremental piece: EIG's 0.73-point collapse when you take the graph away and TreeMem's 11-point drop when you remove tree credit are both confirmations of what the field already suspected — shared state without governance is a liability.

The genuinely new piece: CAFE inverting the quality-degradation ranking. The architecture that degrades most under stress also produces the richest learnable signal. That is a design principle worth carrying forward. Resilience hides the problem.

We would watch two things. Whether CAFE's detected antifragility-compatible regimes actually convert into performance gains once you close the loop and adapt. And whether governed memory's ratification bottlenecks remain tractable when the operator pool expands from one human to many who disagree.

The hallucination wasn't the problem. The reward wasn't the signal. The transcript wasn't the idea. Today's papers keep pulling hidden structure into the light.
