## Themes

**Context is the hidden tax on every agent.** Tool Attention Is All You Need measures it at 50k tokens per turn re-serializing schemas the model has already seen. Empirical Comparison of Agent Communication Protocols pins the crossover where accumulated context flips from asset to liability. Learning to Communicate sidesteps the problem entirely by replacing text handoffs with latent traces. All three converge on one point: the bottleneck isn't reasoning power, it's what you force into the window.

**The serial pipeline is optional.** Speculative Actions borrows from CPU architecture to pre-execute likely next steps in parallel. Learning to Communicate rewrites the inter-agent channel so gradients flow instead of crashing into discrete tokens. Both reject "call → wait → call → wait" as destiny — and both show measurable gains from breaking it.

**MCP's text surface is both wasteful and weaponizable.** Two papers expose MCP's stateless, text-first design from opposite flanks. Tool Attention shows the protocol wastes orders of magnitude more tokens than it needs. Breaking MCP with Function Hijacking Attacks shows those same unguarded text descriptions are an adversarial surface — 60 poisoned tokens hijack an agent nearly 100% of the time. The simplicity that makes MCP easy to adopt makes it fragile.

**Delegation beats accumulation past a tipping point.** Empirical Comparison of Agent Communication Protocols finds a clean crossover: single-agent MCP wins on simple queries, but A2A's context-isolated specialists take over when tasks get complex. Learning to Communicate mirrors this — each agent writes its own KV segment rather than overwriting shared state. When one window can't hold the context, you split it.

**Small levers move big systems.** Two hundred ten SFT samples push AIME accuracy up 27 points. A zero-cost keyword router tracks the winner at every complexity tier. Sixty adversarial tokens hijack an agent with near-perfect success. Intent-conditioned gating slashes token spend by 95%. The control surfaces in agentic systems are shockingly small — and surprisingly powerful.

## Editorial

Five papers, one thesis by accident. Every one of them is about what happens when you stop pretending the context window is infinite.

Tool Attention measures the waste and cuts it by 95%. The protocol comparison shows the waste tipping into failure and prescribes a router. DiffMAS replaces the text pipe with a latent trace. Speculative Actions sidesteps the serial bottleneck that creates the waste in the first place.

And Breaking MCP with Function Hijacking Attacks is the paper that should keep infrastructure teams awake. The same flat text descriptions that bloat context are an adversarial surface — wasteful and dangerous is quite the double.

The genuine architecture changes are DiffMAS and Speculative Actions: gradient-bearing traces and speculative prefetching rewrite how agents communicate and execute. Tool Attention is effective but incremental — the downstream success numbers are projections from token counts, not live task runs. The protocol comparison is solid empirical work with a clean crossover story.

The discipline emerging from all of this is context engineering. Everything else is plumbing.
